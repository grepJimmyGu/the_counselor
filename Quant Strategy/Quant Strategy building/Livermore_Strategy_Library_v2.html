1. Your decisions & framing

Scope
Stock quant trading only. ETF strategies in scope when ETF is an equity proxy (SPDR sectors, factor ETFs).
Time horizon
Mix of all. Each template tagged. Intraday deferred behind data-plan upgrade.
Universe
Full universe + sector ETFs. Engine generalizes momentum_rotation into a reusable cross-sectional helper.
Source ranking
All sources, ranked by robustness. Each template ships with evidence tier (A/B/C) and capacity tag (retail/prosumer/institutional).
Deliverable
Specs + prompts. Prompts in §11 designed for fresh agent sessions.

2. Livermore audit — what exists today

Engine capability

Daily-bar long-only backtester. Single-asset signal path and cross-sectional ranking path (momentum_rotation) both work today.

Schema (StrategyJSON)

Enum-locked strategy_type with 6 values, generic rules array, position_sizing (equal/fixed), risk_management, cash_management.

Services available

market_data, sentiment_service, fundamental_service, screener_service, sec_edgar_client, robustness_service. Big unlock: non-price strategies achievable without new vendor contracts.

Constraints

Long-only. No shorts. No options. No intraday data. universe is a list, not a dynamic screen.

Most MVP templates fit current schema with additive changes. v2 (fundamental, event) requires wiring existing services into engine. v3 (sentiment, ML) needs a SignalProvider abstraction.

4.4 SignalProvider abstraction (v2 foundation)

class SignalProvider(Protocol):
  async def get_signal_frame(
      self, db: Session, symbol: str, start: date, end: date,
      signal_name: str, lookback_days: int = 0,
  ) -> pd.Series: ...

# Concrete providers wrap existing services:
class FundamentalSignalProvider(SignalProvider): ...
class SentimentSignalProvider(SignalProvider): ...
class EventSignalProvider(SignalProvider): ...
class InsiderSignalProvider(SignalProvider): ...
Why this matters. Without point-in-time alignment, fundamental strategies look great in backtest but cheat by using restated financials. The SignalProvider contract makes "as-of" the default.

5. Library: existing strategies (recap)

Moving-Average Filter

Hold when price > N-day MA, else cash. Classic trend filter.

Moving-Average Crossover

Long when fast MA > slow MA, else cash. Standard SMA(50)/SMA(200) baseline.

Momentum Rotation

Rank a small universe by total return; rebalance into top-N.

RSI Mean-Reversion

Buy when RSI < entry; exit when RSI > exit.

Breakout (Donchian)

Long on N-day high breakout; exit on M-day low.

Static Allocation

Fixed weights, periodic rebalance. Useful as benchmark.

6. Library: MVP new templates (price-only)

1. Cross-Sectional Momentum (12-1)
Thesis
Stocks with highest 12-month returns (skipping last month) continue to outperform. Jegadeesh-Titman 1993; replicated thousands of times.
Signal
Rank by return from t-252d to t-21d. Long top decile. Equal-weight.
Parameters
formation_period_days=252, skip_period_days=21, top_n=10
Expected metrics
Sharpe 0.4–0.8 long-only. DD 30%+ during momentum crashes.
2. Time-Series Momentum (Trend)
Thesis
Asset's own past 12m return predicts next 1m return. Moskowitz-Ooi-Pedersen 2012.
Signal
Long when trailing 12-month return > 0, else cash. Optional vol-target.
3. Short-Term Reversal (1-Week)
Thesis
One-week winners underperform losers next week (Lehmann 1990). 2025: strongest in high-MAX stocks.
Caveat
High turnover — require slippage_bps ≥ 10.
4. Pairs Trading (Z-Score)
Thesis
Cointegrated stocks revert to long-run spread. 2024-25 research confirms but pair half-life ~ 1–2 years.
Long-only constraint
Hold long leg-a only when spread says leg-a is cheap; exit to cash otherwise. True pair-neutral is future.
5. Sector Rotation (SPDR ETFs)
Thesis
Cross-sectional momentum on 11 SPDR sector ETFs (XLK, XLF, XLE, XLV, XLY, XLP, XLU, XLI, XLB, XLRE, XLC).
Why ship separately
High-signal, low-noise version of cross-sectional momentum. Recommended default prosumer template.
6. Dual Momentum (Antonacci)
Signal
Pick top by 12-month return; if positive hold, else go to safe asset (bonds/cash).
7. Low Volatility (Min-Vol Tilt)
Thesis
Low-vol stocks have higher risk-adjusted returns (Frazzini-Pedersen BAB, Soebhag-Baltussen-van Vliet 2025).
Signal
Long bottom decile of trailing 252-day volatility.
8. Bollinger Band Mean Reversion
Signal
Buy when close < (MA − k·σ), exit when close > MA.
9. Volatility-Target Overlay
Type
Sizing overlay, applicable to any template. Scales position to hit target annualized vol.

7. Library: v2 templates (fundamental + event)

10. Value Composite (FCF yield + P/B + EV/EBITDA)
Signal
Z-score each metric; average; long top-decile composite.
Why composite
Robeco/AQR: multi-metric value beats single-metric by reducing model risk.
11. Quality (Piotroski F-Score)
Signal
9 binary financial signals; long F-Score ≥ 8. Pair with value pre-filter (P/B < median).
Evidence
Original 1976-96: +7.5% annual outperformance. 20-yr OOS: ~11% / yr.
12. Buyback Yield
Signal
Long top-decile of net buyback yield (TTM). Pair with value gate.
Evidence
Cheap buybacks: 15.3% / yr vs expensive: 5.1% / yr (S&P Global).
13. PEAD (Post-Earnings Drift)
Signal
After earnings, compute SUE; long top-decile for 60 trading days.
Evidence
Garfinkel-Hribar-Hsiao 2024: 5.1% 3-mo risk-adjusted (~20% annualized).
14. Earnings Estimate Revisions
Signal
Long largest positive 90-day change in consensus next-FY EPS. Monthly rebalance.

8. Library: v3 templates (sentiment + composite ML)

15. News Sentiment Momentum

Caveat

16. Insider Buying Signal
Signal
Cluster Form-4 buys in trailing 30 days. Reuse sec_edgar_client.
17. Multi-Factor Composite (Value + Mom + Quality + Low-Vol)
Why flagship
Most defensible long-only equity quant strategy in academia and practice. Pro template.

12. QA checklist before each merge

CheckWhy it mattersHow
Survivorship biasCurrent-membership universe overstates returns 1–3% / yr.SignalProvider must use point-in-time disclosure_date.
Lookahead biasRestated financials cheat.Use as-of disclosure_date or report_date + lag.
Realistic costsHigh-turnover strategies fail after costs.Default slippage 10 bps; warn if user reduces.
CapacityStat-arb / reversal degrade at scale.Show "estimated max AUM" for high-turnover templates.
EvidenceDon't overstate Sharpe from cherry-picked windows.Show literature-derived "typical Sharpe range".
