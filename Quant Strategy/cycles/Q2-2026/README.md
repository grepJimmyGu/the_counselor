# Cycle Template

Copy this entire folder to `cycles/<QX-YYYY>/` at the start of each quarter. Each file has a one-line purpose at the top; fill them in as you go.

```
_template/
├── README.md                       ← this file
├── 01-collect-telemetry.md         ← STEP 1: Livermore usage data
├── 02-research-scan.md             ← STEP 2: KB refresh + market trend scan
├── 03-template-proposals.md       ← STEP 3: 2–3 candidate templates
├── 04-implementation-plan.md       ← STEP 4: prompts, gates, schedule
└── cycle-summary.md                ← END: what shipped, what learned
```

Each step has a 1-sentence definition of done (DoD) listed in the file. Don't skip steps; if a step has nothing new, mark it explicitly so the next cycle can see the gap.
