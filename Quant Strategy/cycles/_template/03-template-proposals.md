# Step 3 — Template Proposals (2–3 new)

**Purpose**: From the telemetry gaps (Step 1) and the research scan (Step 2), propose 2–3 specific new templates worth adding to Livermore.

**DoD**: 2–3 candidate template files created in `templates/candidate/*.md` using `templates/_template-spec.md`, plus a 1-paragraph rationale per template here.

## Proposal funnel — how to choose 2–3 from many ideas

Score each candidate against five questions (1–5):

1. **Demand** — does Step 1 telemetry indicate users wanted this?
2. **Evidence** — is the supporting research Tier A/B (not C)?
3. **Fit** — does it fit Livermore's current engine, or does it need new infra?
4. **Capacity** — does it work at retail/prosumer scale, or institutional-only?
5. **Differentiation** — does Livermore already have a near-equivalent template?

Pick the top 2–3 by total score. Anything with `evidence ≤ 2` is auto-deferred.
