# Strategy Deep Dives

This subfolder holds **per-strategy** notes that synthesize multiple sources (books + papers + market research) into a single canonical document. One file per strategy, named with the same slug as the template lifecycle files (`templates/<status>/<slug>.md`).

Use this folder when a strategy is discussed across many sources and you want one place to consolidate the evidence. Use `books/`, `papers/`, `market-research/` for raw single-source notes.

Front-matter and structure: same as `_kb-entry-template.md` with `source_type: strategy`.
